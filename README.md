# 🚢 Titanic Survival Analysis | Machine Learning

## 📌 Problem Statement
The objective of this project is to analyze passenger data from the Titanic dataset and build a machine learning model to predict whether a passenger survived or not.

---

## 🎯 Objectives
- Perform exploratory data analysis (EDA) to understand survival patterns
- Clean and preprocess the dataset (handle missing values, encoding)
- Build classification models to predict survival
- Evaluate model performance using standard metrics

---

## 📊 Data Overview
- Dataset: Titanic Dataset
- Key Features:
  - Age, Sex, Passenger Class (Pclass)
  - Fare, Embarked, SibSp, Parch
- Target:
  - Survival (0 = No, 1 = Yes)

---

## 🧹 Data Preparation and Processing
- Handled missing values (Age, Embarked, Cabin)
- Converted categorical variables into numerical format
- Performed feature selection and scaling (if applicable)
- Split dataset into training and testing sets

---

## 🤖 Machine Learning Models Used
- Logistic Regression
- Decision Tree Classifier
- Random Forest Classifier

---

## 📈 Model Evaluation
- Achieved ~75–85% accuracy (depending on model)
- Evaluation metrics used:
  - Confusion Matrix
  - Precision
  - Recall
  - F1-Score

---

## 📊 Visualizations
- Survival rate by gender and class
- Age distribution
- Correlation heatmap
- Count plots for categorical features

---

## 🔍 Key Insights
- Female passengers had a higher survival rate than males
- First-class passengers had better survival chances
- Younger passengers had slightly higher survival probability
